<objective>
Fix the bg-paper CSS transparency bug so the sticky header has an opaque background when scrolled.

Purpose: The header uses `bg-paper` as its background class. In `tailwind.config.ts` the paper colour is defined as `hsl(var(--paper))`. But in `globals.css`, `--paper` is stored as `hsl(40 20% 98%)` — a full HSL call. This produces `hsl(hsl(40 20% 98%))` which is invalid CSS; browsers render it as transparent. Fixing `globals.css` to store only the channel values (`40 20% 98%`) makes the resolved value valid: `hsl(40 20% 98%)`.

Output: One-line change in globals.css for both light and dark --paper declarations. Header becomes opaque immediately.
</objective>
