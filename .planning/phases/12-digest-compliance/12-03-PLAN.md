**Create lib/referral.ts** with the following exports:

1. `generateCode(): string` — returns `crypto.randomBytes(4).toString('hex')` (8 hex chars).

2. `getOrCreateReferralCode(userId: string): Promise<string>` —
   - If Redis available: check `referral-code:{userId}`. If exists, return it. Otherwise generate a new code, SET `referral-code:{userId}` to code AND SET `referral:{code}` to userId (bidirectional mapping), return code.
   - If no Redis (dev): return deterministic `crypto.createHash('sha256').update(userId).digest('hex').slice(0, 8)`.

3. `recordReferral(newUserId: string, referralCode: string): Promise<{ rewarded: boolean }>` —
   - If no Redis, return `{ rewarded: false }`.
   - Look up referrer: `referral:{referralCode}` -> referrerId. If not found or referrerId === newUserId (self-referral guard), return.
   - SET `referred-by:{newUserId}` -> referrerId with NX (prevent double-counting). If SET returned null (already exists), return.
   - INCR `referral-count:{referrerId}`.
   - If `newCount % 3 === 0`, call `applyFreeMonthCoupon(referrerId)` and return `{ rewarded: true }`.
   - Otherwise return `{ rewarded: false }`.

4. `applyFreeMonthCoupon(referrerId: string): Promise<void>` (internal, not exported) —
   - Import Stripe, create client.
   - Look up referrer's subscription via `getSubscription(referrerId)` from lib/subscription.ts.
   - If no subscription or no stripeSubscriptionId, log warning and return.
   - Create coupon: `stripe.coupons.create({ percent_off: 100, duration: 'once', max_redemptions: 1, name: 'Referral reward - 1 free month' })`.
   - Apply: `stripe.subscriptions.update(subId, { discounts: [{ coupon: coupon.id }] })`.
   - Log success.

Use the same `useRedis()` / `getRedis()` pattern as lib/subscription.ts (inline require of @upstash/redis).

**Create components/ReferralTracker.tsx** — a `'use client'` component:
- Uses `useSearchParams()` from `next/navigation`.
- In `useEffect`, reads `ref` param. If present, sets `document.cookie = \`folio-ref=${ref}; max-age=${30 * 24 * 3600}; path=/; SameSite=Lax\``.
- Renders `null` (invisible component).
- Wrap `useSearchParams()` usage in a `<Suspense>` boundary (Next.js 15 requirement for useSearchParams in client components).

**Update app/layout.tsx** — add `<ReferralTracker />` inside the body, after existing providers. Wrap in `<Suspense fallback={null}>` if the component itself doesn't handle it internally.
